
# SAE 2.01 PARTIE 2 O'De France
Deuxième partie de la SAE 2.01 avec la création d'une application pour l'entreprise O De France.
Projet à utiliser avec GitLab UnivArtois, en utilisant le langage de programmation Java.
Dans ce projet, nous utilisons ces notions :
- Test Unitaire -> JUnit
- Build Tool -> Graddle
- Editeur FXML -> Scene Builder
- Vue -> JavaFX
- Partage projet -> GitLab
